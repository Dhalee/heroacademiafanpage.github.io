Quirk Database

A quirk is a special superhuman ability that a person can have. About 80% of the population has one.

Quirk Types

Emitter Type: Quirks that allow the user to generate and control something.

Transformation Type: Quirks that allow the user to change their body's form.

Mutant Type: Quirks that cause a permanent change to the user's body.
